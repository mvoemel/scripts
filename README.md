# Scripts

This Repository holds different scripts used for different scenarios.

## Converters

- `.mermaid` to `.png` -> [Mermaid to PNG Converter](./mermaid_to_png_converter.html)
- split, delete rearange pages in `.pdf`s -> [PDF Page Manager](./pdf_page_manager.html)
- stress test a website -> [Website Stress Tester](./website_stress_test.py)
- download yt videos in 4k -> [YT Video Downloader](./yt_vid_downloader.py)
